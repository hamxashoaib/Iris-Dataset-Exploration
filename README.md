# Iris Dataset Exploration and Visualization

## Project Overview

This project is part of an AI/ML Internship program and focuses on Exploratory Data Analysis (EDA) using the Iris Dataset. The objective is to understand the structure of the dataset, examine feature distributions, identify relationships between variables, and gain meaningful insights through visualization.

---

## Objective

The primary goals of this task are:

* Load and inspect a dataset using Python and Pandas.
* Perform exploratory data analysis (EDA).
* Generate descriptive statistics.
* Visualize feature relationships and distributions.
* Draw insights from the dataset.

---

## Dataset

### Iris Dataset

The Iris dataset is one of the most widely used datasets in machine learning and statistics.

#### Features

* Sepal Length (cm)
* Sepal Width (cm)
* Petal Length (cm)
* Petal Width (cm)

#### Target Variable

* Species

  * Setosa
  * Versicolor
  * Virginica

#### Dataset Size

* Total Samples: 150
* Features: 4
* Classes: 3

## Key Findings

* The dataset contains no missing values.
* All classes are balanced with equal representation.
* Petal-related features provide the strongest separation between species.
* Setosa is clearly distinguishable from the other two species.
* Strong positive correlation exists between petal length and petal width.
* The dataset is well-suited for classification tasks.

---

## Results

The exploratory analysis successfully revealed patterns, feature relationships, and class separability within the Iris dataset. The findings demonstrate why this dataset is commonly used for introductory machine learning and classification problems.
